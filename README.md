# quant-lab

Quant Lab is a personal quantitative research platform built in Python to design, test, and simulate systematic equity trading strategies.

## Features
- Backtesting engine with rebalancing and transaction cost modeling  
- Point-in-time data alignment to prevent lookahead bias  
- Factor-based stock ranking (profitability, reversal, volatility)  
- Portfolio construction and performance tracking  
- Live simulation pipeline with trade execution via Alpaca paper trading  

## Technologies
- Python (Pandas, NumPy)  
- Financial data APIs (Tiingo, FMP)  
- Local data storage (parquet)  

## Purpose
This project is designed to explore quantitative investment strategies and develop practical experience in financial data analysis and modeling.
